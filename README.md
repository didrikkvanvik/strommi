# Strømmi

Strømmi er en enkel side for å se prisen på strøm i Norge.

![Strømmi Screenshot](./src/assets/strommi.png)

## Features
- View electricity prices by date
- Interactive chart showing price variations
- Color-coded price indicators
- Responsive design

## Tech Stack
- Vue 3
- TypeScript
- Tailwind CSS
- Chart.js
- Vue Query

## Development

```sh
npm install
npm run dev
```
